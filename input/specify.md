# プロダクト名
EmojiLink (絵文字だけでつながるSNS)

# 目的
- 言語を超えてコミュニケーションできる場を作る
- 投稿もリアクションも絵文字のみで完結するシンプルな体験を提供する

# ペルソナ
- プログラミング未経験の高校生が自分の気分を共有したい
- 見て楽しい、触ってすぐ使えるシンプルなSNSを探している

# 必須機能
- タイムラインに投稿を時系列で表示する
- 投稿は1〜3個の絵文字のみ（テキスト不可）
- 投稿への絵文字リアクション（数種類を用意）
- 投稿作成モーダル（絵文字ピッカー付き）
- ローカルJSONでデータを保持

# 画面のざっくりイメージ
- ヘッダー：タイトル、テーマカラー、簡易プロフィール
- タイムライン：カード表示、投稿者アイコン（仮）、投稿絵文字、リアクションボタン
- 投稿モーダル：絵文字ピッカーと送信ボタンのみ

# 技術の希望
- Node.js + Vite
- スタイルはシンプルで高校生が真似しやすいもの

# デモ条件
- ローカルで `npm install && npm run dev` 相当で起動できる
- 少量のデモ投稿が最初から表示される
